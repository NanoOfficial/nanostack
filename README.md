# Installation:
## Pre-requirements:
- Operating System[macOS/Linux/Windows]
- 4+ GiB Ram
- 10+ GiB SSD
- latest version of **rust** if you dont have it download the latest version of rust from [here](https://www.rust-lang.org/tools/install)

## Installation:
```
cargo install 
```
